Файл Delivery.txt в папке проекта Delivery содержит некорректные входные данные в формате CSV для проверки валидации.
В папке проекта Delivery присутствует файл с корректными входными данными correctDelivery.txt
Файл _deliveryOrder.txt содержит заказы, которые подходят под требованию о времени. Этот файл создаётся автоматически, если не указано имя файла с отфильтрованными заказами во время работы программы. 
Файл _deliveryLog.txt содержит лог работы программы. Этот файл создается автоматически,  если не указано имя файла с логами во время работы программы.
